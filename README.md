# OLIVA Project

Global fitness SaaS platform.

## Structure

- **/client**: React + Vite SPA (Frontend)
- **/server**: Node.js + Express + PostgreSQL (Backend)

## Getting Started

### Backend
1. Go to `server/`
2. `npm install`
3. Set up `.env` (see `server/README.md`)
4. `npm run db:init`
5. `npm run dev`

### Frontend
1. Go to `client/`
2. `npm install`
3. `npm run dev`
